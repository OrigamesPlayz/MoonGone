# MoonGone
This is a game where you are stuck on the moon with some red enemies and your space shuttle has crashed!

Find your way back to Earth by collecting fuel by killing the enemies. Good luck!

Controls:

WASD - To move

Left Click - Shoot gun

E - Pick up fuel rocks

F - Enter space shuttle

This game is made for Siege Week 11 so not much effort but time well spent have fun reviewers and other :).
